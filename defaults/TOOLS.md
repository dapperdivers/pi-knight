# Knight Tools

## Built-in Tools
- read: Read file contents
- write: Create or overwrite files
- edit: Make precise edits to files
- bash: Execute shell commands

## Vault Access
- Read: /vault (shared Obsidian vault)
- Write: /vault/Briefings/ and /vault/Roundtable/ only

## NATS
- Results are published automatically after task completion
- Use nats-comms skill for out-of-band messaging
